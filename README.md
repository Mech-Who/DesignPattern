# Design Pattern

## 一、项目介绍

主要用来学习设计模式的理论，并通过简单的练习进行理论知识的实践。
同时使用`C++`进行实现，锻炼自己的`C++`能力。

## 二、主要内容

1. 创建型模式（5种）：主要用来创建对象
	- 工厂方法模式（`FactoryMethodPattern`项目）
	- 抽象工厂模式（`AbstractFactoryPattern`项目）
	- 生成器模式（`BuilderPattern`项目）
	- 原型模式（`PrototypePattern`项目）
	- 单例模式（`SingletonPattern`项目）
2. 结构型模式（7种）：主要用于组合使用不同对象
	- 适配器模式（`AdapterPattern`项目）
	- 桥接模式（`BridgePattern`项目）
	- 组合模式（`CompositePattern`项目）
	- 装饰模式（`DecoratePattern`项目）
	- 外观模式（`FacadePattern`项目）
	- 享元模式（`FlyweightPattern`项目）
	- 代理模式（`ProxyPattern`项目）
3. 行为型模式（10种）：主要实现不同对象之间的交互
	- 责任链模式（`ResponsibilityChainPattern`项目）
	- 命令模式（`CommandPattern`项目）
	- 迭代器模式（`IteratorPattern`项目）
	- 中介者模式（`MediatorPattern`项目）
	- 备忘录模式（`MementoPattern`项目）
	- 观察者模式（`ObserverPattern`项目）
	- 状态模式（`StatePattern`项目）
	- 策略模式（`StrategyPattern`项目）
	- 模板方法模式（`TemplateMethodPattern`项目）
	- 访问者模式（`VisitorPattern`项目）
